# Adapters

Adapters connect Trade Ops to external systems.

Examples:

- TradingView
- Alpaca
- Kalshi
- Polymarket
- Jupiter
- Polygon
- RugCheck
- Binance Futures
- Hyperliquid
- Deribit
- Coinbase
- Kraken
- DeFiLlama
- FRED
- BLS
- Treasury FiscalData
- SEC EDGAR

Each adapter should expose a stable internal interface even if the underlying implementation changes.
