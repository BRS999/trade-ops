# Wiki

The wiki is local compiled trading memory.

Do not commit personal symbol notes, setup stats, edges, mistakes, market context, or live lessons from actual paper/live trades.

Tracked wiki content should be limited to:

- this README
- `wiki/market/README.md`
- scrubbed examples under `wiki/examples/`

Local-only working memory commonly lives at:

- `wiki/edges.md`
- `wiki/mistakes.md`
- `wiki/symbols/`
- `wiki/setups/`
- `wiki/market/context.md`
